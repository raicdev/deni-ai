## Tech Stack

- **Frontend**: Next.js, React, TypeScript
- **AI SDK**: Vercel AI SDK
- **Styling**: Tailwind CSS
- **Backend**: Next.js API Routes
- **Authentication**: Firebase Authentication
- **Database**: Firebase Firestore
- **Deployment**: Vercel (recommended)
